== Description ==

[Product Pricing Table plugin](https://woobewoo.com/plugins/woocommerce-pricing-table/ "Product Pricing Table WordPress plugin") by WooBeWoo is a best tool for making your goods to look in a favorable light. Colorful and compact tables will bring novelty to your site, and let you show your products benefits to your clients. The only thing they need to do is to compare products, choose your best offer and click on "PURCHASE" button! Just couple of quick steps - pick product pricing table template, add content and you're ready with your first table.
It's never been so easy to create and manage pricing and comparison tables . User friendly table builder was created by our coders to make the plugin easy-in-use as for skilled developers, as for people without any programming skills. Your comparison table can be customized from head to toe via special interface, or you can use built-in CSS Editor if you have some CSS knowledges.

== Installation ==

a) VIA FTP
Download zip archive with plugin
Extract files
Copy the folder with plugin
Open via ftp folder \wp-content\plugins\
Paste plugin folder into here
Go to admin panel => Plugins => press activate

B) DIRECT UPLOAD VIA ADMIN AREA
Download zip archive with plugin
Go to admin panel => Plugins => Add new
Choose the archive with plugin and upload it
Wait until you get the notification installation went successfully
Press Activate button

C) VIA WORDPRESS REPOSITORY
Navigate to admin panel => Plugins => Add new
Type "Woocommerce Pricing Table by WooBeWoo" in the search field
Choose our plugin from the list and click "Install"
Wait until you get the notification installation went successfully
Press Activate button

= My Product Pricing Table Plugin is not working. Why? =

If you have problems with the work of plugin or something is not working as it should, first follow this follow steps:
*Test the plugin with a WordPress default theme, to be sure that the error is not caused by the theme you are currently using.
*Deactivate all plugins you are using and check if the problem is still occurring.
*Ensure that you plugin version, your theme version and your WordPress are updated and the problem has not already been solved in a later plugin update.
If none of the this advises help you, contact us via [support form on our website](https://woobewoo.com/contact-us/) and describe your problem.
